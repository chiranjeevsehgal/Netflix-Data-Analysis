# Netflix Data Analysis
 
Objective of this analysis is basically to take out insights from the dataset and visualise it to observe trends. In this project, we are going to process and analyse the data using Numpy and Pandas libraries and then we will visualise it using matplotlib library.

This analysis basically involves:

- Preparing the data for the analysis.
- Rating measure of movies/shows.
- Number of releases in last 5 years.
- Number of movies/shows.
- Top 5 countries which produced highest number of titles.
- 5 most popular rating.
- 5 top rated genres.
- 5 least rated genres.
